## 预测模型
对于大装甲板和小装甲板分别训练了两个简单的图像分类模型

小装甲板: 64 * 64 * 1  
大装甲板: 64 * 128 * 1

一些测试结果:  
类别编号详见装甲板贴纸素材包  
样本0为小装甲板数字3; 样本4,5,6为大装甲板数字1,3,4; 其余样本为噪声
